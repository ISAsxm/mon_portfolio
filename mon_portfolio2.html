Réalisations.

projet site onepage:

Projet d'intégration web qui consistait à produire un site internet onepage, thème libre, à partir d'un template. Pour mener à bien ce projet, j'ai tout d'abord réalisé un wireframe de la maquette afin de shématiser ma structure HTML. Une fois la maquette analysée et les différents éléments repérés, j'ai réalisé mon fichier HTML pour ensuite coder le fichier CSS afin de mettre en forme la structure et les animations en respectant les exigences du template. La mise en page terminée, j'ai intégré des scripts codés en JavaScript pour le dynamisme et les évènements suite aux différentes actions des utilisateurs(lors du scroll, au clic sur les bouttons, lors du slide pour la partie blog).

projet blog:

Projet orienté Back-end, dans lequel j'ai utilisé des requêtes AJAX, du JavaScript et JQuery côté client, PHP et MySql côté serveur en me servant de JSON pour communiquer entre les deux. Pour la réalisation de ce blog, j'ai tout d'abord orienté le projet vers la partie back-end, en codant les scripts nécéssaires pour l'échange de données. Puis j'ai commencer à développer les fichiers qui me permettrait de faire un C. R. U. D. Une fois les scripts opérationnels, j'ai porté mon intérêt côté Front-end et ai réalisé la page d'accueil de ce blog en utilisant Bootstrap pour la structure HTML/CSS et AJAX/JQuerry pour l'affichage des données et le dynamisme des animations.
